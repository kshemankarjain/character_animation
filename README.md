## General info
This project is simple endless runner game which works on general movement keys.
	
## Technologies
Project is created with:
* visual studio c# v2019
* unity 3D v2020.1.0a12
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
